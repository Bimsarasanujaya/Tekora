Tekora V28 Clean Brand + Password Reset

- Removed visible version labels such as V24/V25/V26 from user-facing UI.
- Removed Phase 1 Core and EXE later strip from the dashboard.
- Added professional forgot-password flow with secure reset tokens.
- Reset links can be emailed through SMTP environment variables.
- If SMTP is not configured, reset links are saved to data/email_outbox.json for testing.

SMTP variables for Gmail:
TEKORA_SMTP_HOST=smtp.gmail.com
TEKORA_SMTP_PORT=587
TEKORA_SMTP_USERNAME=yourgmail@gmail.com
TEKORA_SMTP_PASSWORD=your Gmail App Password
TEKORA_SMTP_FROM=Tekora <yourgmail@gmail.com>
