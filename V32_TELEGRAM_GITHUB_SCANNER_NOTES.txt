How it works:
1. GitHub Actions runs every 5 minutes.
2. If there is no active trade, Tekora scans 5m/15m scalp/aggressive setups.
3. If a setup score is 80+ and not HIGH RISK, it sends the VIP signal to Telegram.
4. It stores that setup in state/telegram_state.json.
5. On later cycles, it monitors entry/TP/SL using live MEXC candles.
6. It sends Telegram updates when status changes.
7. After TP3, SL, or expiry, it clears active_trade and scans again.
